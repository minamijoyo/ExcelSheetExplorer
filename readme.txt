1.はじめに
Excelファイルを開かずにシート名の一覧を閲覧できるビューア(検索機能付)です。
Excel97-2003で用いられているBIFF8形式のxlsファイルに対して、バイナリデータを
直接解析しシート名を取得します。
Excelコンポーネントを経由してオブジェクトを読み込まないためVBAよりも高速に動作します。

3.インストール
実行にはMicrosoft社の.NET framework 2.0のランタイムが必要です。
各自適当にMicrosoftからダウンロードしてインストールして下さい。
http://www.microsoft.com/downloads/details.aspx?familyid=0856EACB-4362-4B0D-8EDD-AAB15C5E04F5&displaylang=ja

本ソフト自体はインストール不要です。
適当なフォルダに解凍してExcelSheetExplorer.exeを実行してください。

WinXP SP2 + .NET2.0 + Excel2000で動作確認しています。

4.使い方
ヘルプボタンを押すと使い方の説明(help.txt)がテキストエディタで起動します
詳しくはそちらを参照して下さい

5.アンインストール
解凍したフォルダごとゴミ箱に捨ててください。
レジストリなどはいじっていません。
.NET frameworkは他のソフトに不要ならアンインストールして下さい。

6.バージョン履歴
2007/08/18 v0.4β
シート名を取得する際の解析ロジックを一部変更。
シート名取得失敗したファイルを検索結果に含めるオプションを追加。

2007/06/09 v0.3β
検索機能を追加。
表記ゆれに対処するため、部分一致検索で大/小文字の違いを無視し、
スペース区切りでOR検索できるようにした。
いくつかのランタイムエラーのバグ修正。

2007/05/26 v0.2β
ツリービューをやめて、エクスプローラから直接フォルダをドラッグ＆ドロップできるようにした。
これに伴いローカルのみではなく共有フォルダなどもアクセスできるようになった。
アイコンの埋め込みや使い方ボタンを作った。
前回終了時のフォルダを次回起動時にデフォルトで開くようにした。
